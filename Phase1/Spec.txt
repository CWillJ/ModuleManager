Module Manager Spec

Modules:
   - Needs to look for previous saves on start up and load them if found.
   - Saves need to be saved/loaded based on user
   - Button to look for dll's in a specified file location
   - Button to save the configuration of enabled/disabled modules
   - Something that looks through a file location for all dll's in the location.
   - It needs to find the dll's and the associated xml file to access the info.
   - Need to display method properties via one way binding


UI:
   - Load button
   - Save button
   - Empty area before loading a file location with dll's
   - Enable/disable check box for each module found
   - Methods displayed under module name
   - Description text box that shows selected method's properties
